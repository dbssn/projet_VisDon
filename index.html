Notes :

  • Source : InsideAirbnb, (2020, 13 Juin). Detailed Listings data for Barcelona. Inside Airbnb : Adding data to the debate. Consulté le 30.08.2019. | Nom du fichier : "listings.csv.gz"
  • Les données reflètent l'état au 13 juin 2020, il s'agit d'un aperçu des annonces à un instant donné.
  • Les prix sont indiqués en USD pour une personne et pour une nuit, frais de ménage inclus. Les frais de la plateforme ne semblent pas être inclus dans les données utilisées.
  • La formule utilisée pour le calcul du prix est la suivante : (prix du logement + frais de ménage) / nombre d’hôtes incluses pour ce prix. Les prix indiqués ne devraient donc pas être extrapolés pour 2, 3, …, n personnes.
  • Seuls les quartiers pour lesquels plus de 100 annonces existent sont considérés (soit 25 quartiers).
  • Les données traitées ici portent sur des séjours de courte durée. Les annonces pour lesquelles un minimum de plus de 7 nuits était exigé ont été écartées.
